# Anonymous Cloud Notes

**A minimalist, end-to-end encrypted notepad.** No accounts, no tracking, no bullshit.

Create a note → get a secret link. Anyone with the link can view and edit it. Notes are encrypted in your browser using **AES-256-GCM** before they reach the server.

### Features

- True end-to-end encryption (server never sees plaintext)
- Fully anonymous — no signup or email required
- Real-time collaborative editing
- Self-hosted and open source
- Clean, mobile-friendly interface

### Security

Even if the server is compromised or subpoenaed, your notes remain unreadable without the secret key.

**Try it:** [https://notes.chrisjones.online](https://notes.chrisjones.online)

---

**Self-hosting:** Just drop the files on any PHP 8+ server and point your domain at it.
